概览：SQL Server 2017与SQL Server Management Studio的安装，当初安装时没记录，所以只是一些参考过的链接记录，说实话根本看不懂哪些需要还是不需要，头大。

下载地址

SQL Server 2017：https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads

**SQL Server Management Studio **：https://docs.microsoft.com/zh-cn/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017

安装参考链接

  1. 逍遥小丸子：SQL Server数据库基础：SQL Server 2017安装教程：https://blog.csdn.net/dnruanjian/article/details/85235139

  1. 开山怪-SQL server 2016 安装步骤：https://www.cnblogs.com/ksguai/p/5869558.html

也是2016版本的安装，基本步骤没啥问题。

  1. SQL Server 2017 安装过程中的一点说明（有点意思）：https://blog.csdn.net/a_222850215/article/details/80602723

我的安装

印象中功能选择是那个机器学习服务，关于R和Python的那一部分我是没有勾选的，因为那会感觉对我没啥用，就没安装。

建议安装的时候多百度吧。。。

查询SQLServer安装版本

在Microsoft SQL Server Managment Studio中新建查询，输入 select @@version 然后执行即可看到安装版本。
