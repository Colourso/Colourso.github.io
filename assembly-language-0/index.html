现如今大部分电脑都是64位，没有DEBUG工具了。所以需要安装DOSBox以及DEBUG工具来进入debug模式。

DOSBox软件下载

链接：https://pan.baidu.com/s/1mwh1jF6fGPJ9AGa_vsnpTg
提取码：4cqc

使用教程

下载好软件之后进行解压，这里面的这个DOSBox可以直接点击使用，无需安装，所以最好要记得解压后放置的路径。

找到路径 DOSBox-0.74\masm ，这里面有debug的可执行程序，还有其他后续学习会用到的程序比如link.exe等等。把整个 masm文件夹 拷贝至 D盘或者E盘(任何一个盘符都可以)，这里我放在了E盘。

回到 DOSBox-0.74的路径下，执行 DOSBox.exe 文件。会同时打开两个窗口。

然后进行挂载操作，输入 mount C E:/masm .这里大小写随意，后面的路径就是你放置 masm文件夹 的位置。

然后会弹出Drive C is mounted as local directory e:/masm\这句话，表示挂载成功。其实意思就是把 e:/masm 这里当成了虚拟的C盘来进行使用。

之后输入 c: 进入刚弄好的虚拟的c盘,输入 debug 即可进入debug。

注意事项

使用mount命令之后一定要先进入C盘再使用debug，不然会显示 Illegal command: debug 哦！
