Array index

取出上述矩阵A的特定位置的元素

matlab下标从1开始

  • A(1,2) ——取出矩阵坐标为(1,2)位置的元素。第一行，第二列
  • A(4) ——从第一列数起对应的index值
  • A([1 3 5]) —— 取出第1、3、5个元素的值
  • A([1 3; 1 3]) ——取出第1、3个元素构成一个矩阵
  • A([1 3] , [1 3]) ——前面代表第1、3行，后面代表第1、3列，取出他们构成的交集，然后组成一个矩阵

矩阵与实数做运算

A -- 表示矩阵， a -- 表示实数

方程式求根

symbolic variable

创建symbolic variable的方式有两种：sym或者syms

syms x
%或者
x=sym('x')

x+x+x = 3*x

solve()函数求解

例：y = x * sin(x) - x = 0

syms x
solve('x*sin(x) - x',x)

解多元方程

>> syms x y
>> eq1 = x -2*y -5;
>> eq2 = x + y -6;
>> A = solve(eq1,eq2,x,y)

>> A = 
    x: [1x1 sym]
    y: [1x1 sym]

>> A.x
 
ans =
 
17/3
 
>> A.y
 
ans =
 
1/3
