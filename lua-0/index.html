直接安装

下载

进入网站： http://joedf.ahkscript.org/LuaBuilds/

这里我下载的是 lua-5.3.5_Win64_bin 。解压内容如下所示

配置环境变量

然后存放于一个不会轻易删除的文件夹，不要带中文。复制整个路径，添加环境变量。

  1. 点击环境变量
  2. 点击path，之后点击编辑
  3. 弹出编辑环境变量的窗口，点击新建
  4. 填入刚才复制的路径，点击确定。结束

测试

在cmd输入lua，输出所示：Lua 5.3.5 Copyright (C) 1994-2018 Lua.org, PUC-Rio

然后尝试打出入门语句 Hello World。

存在的问题

貌似执行的时候只能先写好文件，然后去用cmd来执行文件。

lua hello.lua

形如这种形式

使用VS CODE

在VS CODE中安装插件 Lua Debug 。之后要对VS CODE 进行重启。

之后写一个lua文件，然后Ctrl +F5 (在不调试的情况下启动)，即可在vs code的控制台查看到结果。
